## Installation

#### Quick Install

```bash
$ bash <(curl -fsSL https://git.io/fAl8K)
```
#### Manual

```bash
$ git clone git@github.com:starifly/FlyVim.git ~/.FlyVim
$ cd ~/.FlyVim
$ make vim     # install FlyVim for Vim
$ make neovim  # install FlyVim for NeoVim
```
